# The Scout — Guardrails

## Why Guardrails Matter

Without guardrails, an automated outreach agent can:
- Get your LinkedIn account banned
- Look desperate to recruiters
- Waste time on irrelevant jobs
- Send duplicate messages to same HR

The Scout has 10 built-in guardrails to prevent all of this.

---

## Guardrail 1 — Skip Duplicate Jobs

**What it does:** Checks Google Sheets before applying to any job. If the job URL already exists → skip.

**Why:** Sending the same HR the same message twice looks unprofessional.

**How it works:**
- "Duplicate Check" node reads Google Sheets
- "Is Duplicate?" IF node checks if rows returned = 0
- If rows > 0 → route to false branch (skip)
- If rows = 0 → route to true branch (continue)

---

## Guardrail 2 — Rate Limit Delay

**What it does:** Waits 30-60 seconds between each AI API call.

**Why:** Groq free tier has 30 requests/minute limit. Too many requests = error.

**How it works:**
- "Rate Limit Delay" Wait node added before each AI call
- Set to 30-60 seconds depending on your usage

---

## Guardrail 3 — Fresh Jobs Only (7 Days)

**What it does:** Only processes jobs posted in the last 7 days.

**Why:** Applying to 3-month-old jobs wastes your time — position is likely filled.

**How it works:**
- JSearch API URL parameter: `date_posted=week`
- Filters results to last 7 days automatically

---

## Guardrail 4 — Daily Cap (20 Jobs)

**What it does:** Processes maximum 20 jobs per day.

**Why:** Sending 100+ emails/day looks like spam and can get Gmail flagged.

**How it works:**
- "Split Jobs" node batch size set to 20
- Agent stops after 20 jobs regardless

---

## Guardrail 5 — Blacklist Companies

**What it does:** Skips companies you don't want to apply to.

**Why:** You might have had a bad interview experience or know someone who works there.

**How it works:**
- Add company names to "Blacklist" tab in Google Sheets
- Agent checks blacklist before each outreach

---

## Guardrail 6 — Contact Routing

**What it does:** Routes to different actions based on what contact info was found.

**Why:** Different contact methods require different approaches.

**How it works:**
- Has Email → send automated email
- Has LinkedIn (personal) → generate message for manual send
- Nothing found → skip and log

---

## Guardrail 7 — Unique AI Messages

**What it does:** Groq generates a unique personalised message for every single company.

**Why:** If LinkedIn detects identical messages, your account gets flagged as spam.

**How it works:**
- Groq prompt includes company name and job title
- Every message is different in tone and content
- Signed off as Shruthi with personal touch

---

## Guardrail 8 — Status Logging

**What it does:** Logs every action to Google Sheets with status.

**Why:** Full audit trail of what was sent, skipped and why.

**Statuses logged:**
- Sent — email successfully delivered
- Manual — LinkedIn DM ready to send manually
- Skipped — duplicate or no contact found
- Error — something went wrong

---

## Guardrail 9 — Weekdays Only

**What it does:** Schedule set to run Monday to Friday only.

**Why:** Recruiters don't check emails on weekends. Running on weekends wastes API quota.

**How it works:**
- Schedule Trigger configured for weekdays
- Timezone set to Asia/Kolkata (IST)

---

## Guardrail 10 — Human Review for LinkedIn

**What it does:** LinkedIn DMs are never sent automatically — always logged for manual review.

**Why:** LinkedIn actively detects and bans automation. Manual sending is safer and more personal.

**How it works:**
- LinkedIn message generated by AI
- Saved to Google Sheets column H
- LinkedIn profile saved to column F
- You send manually each morning (takes 10 mins)
